字段选项

字段

在本章我们会介绍字段选项和字段类型

字段选项

null

判断数据库是否可以为null，默认不允许。

示例：

models.CharField(null=True)

blank

判断该字段是否可以为空字符串，默认false

示例：

models.CharField(blank=False)

choices

指定select可选内容，参数为列表或元组。

示例：

YEAR_IN_SCHOOL_CHOICES = (
        ('FR', 'Freshman'),
        ('SO', 'Sophomore'),
        ('JR', 'Junior'),
        ('SR', 'Senior'),
    )
models.CharField(choices=YEAR_IN_SCHOOL_CHOICES)

db_column

指定字段在数据库中的名称，默认值为字段名称

示例：

model.CharField(db_column="ColumnName")

db_index

如果为True创建数据库索引

示例：

model.CharField(db_index=True)

db_tablespace

表空间，如果该字段已经创建了索引，那么数据库表空间的名称将作为该字段的索引名。注意，部分数据库不支持表空间。

default

字段的默认值。

示例：

models.CharField(default="fieldname")

editable

如果为false，则不可编辑，那么admin和其它model则忽略掉这个字段。

error_messages

能够重写默认抛出的错误信息。

help_text

将会作为提示信息显示在表单中。

示例：

help_text="Please use the following format: <em>YYYY-MM-DD</em>."

primary_key

如果是True，这个字段是主键

unique

如果是True，该字段在整个表格中必须唯一

unique_for_date/unique_for_moth/unique_for_year

时间日、月、年是否唯一

verbose__name

字段显示名称

validators

字段验证器

from django.core.exceptions import ValidationError
from django.utils.translation import gettext_lazy as _

def validate_even(value):
    if value % 2 != 0:
        raise ValidationError(
            _('%(value)s is not an even number'),
            params={'value': value},
        )

from django.db import models

class MyModel(models.Model):
    even_field = models.IntegerField(validators=[validate_even])
